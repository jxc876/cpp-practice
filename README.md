# About

A project to practice C++ concepts

The examples use the Boost library, so you'll need it installed
* https://www.boost.org/doc/user-guide/getting-started.html

Uses CMake to build & run

```shell
mkdir build
cmake -B build/ .
cmake --build build/
```

Then run the programs:

```shell
./build/main
./build/point
# etc...
```
